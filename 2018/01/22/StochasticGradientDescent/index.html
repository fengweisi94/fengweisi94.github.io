Stochastic Gradient Descent(随机梯度下降)

Stochastic Gradient Descent(SGD) 是一种简单但非常有效的方法，用于在诸如（线性）支持向量机和逻辑回归之类的凸损失函数下的线性分类器的辨别学习。 SGD已成功应用于文本分类和自然语言处理中经常遇到的大规模和稀疏机器学习问题。

随机梯度下降的优点是：

  • 高效
  • 易于实现（很多机会进行代码调优）

随机梯度下降的缺点是：

  • SGD需要一些 hyperparameters(超参数)，如 regularization parameter(正则化参数)和迭代次数
  • SGD对特征缩放敏感

Classification(分类)

SGDClassifier 类实现了一个简单的随机梯度下降学习程序，支持不同的 loss functions (损失函数)和 penalties for classification (分类处罚)。比如 clf=SGDClassifier(loss=”hinge”, penalty=”l2”)

  • 函数fit(X, y) ：模型拟合数据——必须拟合两个 array :保存训练样本的 size[n_samples,n_features] 的 array X 以及保存训练样本的目标值（类标签）的 size[n_sample] 的 array y
  • 函数predict(test_X) ：模型预测新数据
  • 函数predict_proba(test_X) :给出每个样本的概率估计P(y|x),仅限 loss=log、modified_huber
  • 成员coef_ :模型参数
  • 成员intercept_ : 模型截距

具体的 loss function （损失函数）可以通过 loss parameter (损失参数)设定。 SGDClassifier支持以下损失函数：

  • loss=”hinge”:(soft-margin)linear Support Vector Machine(线性支持向量机)
  • loss=”modified_huber”:smoothed hinge loss(平滑的hinge损失)
  • loss=”log”:logistic regression(逻辑回归)
  • and all regression losses below(所有的回归损失)

具体的 penalty （惩罚）可以通过 penalty parameter (惩罚参数)设定。 SGD支持一下惩罚：

  • penalty=”l2”:L2 norm penalty
  • penalty=”l1”:L1 norm penalty
  • penalty=”elasticnet”:Convex combination of L2 and L1 = (1-l1_ratio) L2+l1_ratio L1

默认设置为penalty=”l2”。L1惩罚获得sparse solutions(稀疏解)，将大多数系数降到零。在存在高度相关的属性的情况下， Elastic Net (弹性网)解决了L1惩罚的一些缺陷。参数l1_ratio控制L1和L2惩罚的凸组合。

Regression(回归)

SGDRegressor类实现了一个简单的随机梯度下降学习程序，支持不同的损失函数和惩罚，以适应线性回归模型。 SGDRegressor非常适合大量培训样本的回归问题。
SGDRegressor支持以下损失函数：

  • loss=”squared_loss”:Ordinary least squares
  • loss=”huber”:Huber loss for robust regression
  • loss=”epsilon_insensitive”:linear Support Vector Regression
